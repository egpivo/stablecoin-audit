Stablecoin Audit Evidence Browser

Read-only inspection of completed audit runs

This browser summarizes generated audit evidence. Raw artifacts remain available for verification, but the UI does not create new claims.

It does not claim reserve adequacy, peg stability, redemption capacity, user geography, issuer intent, or actual swap routing.
